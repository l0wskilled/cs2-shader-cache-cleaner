# CS2 Shader Cleaner

A simple, one-click tool to automate the deletion of Counter-Strike 2 and NVIDIA shader caches. 

Sometimes, after a game update or a graphics driver update, clearing old shader files is a recommended troubleshooting step. Doing this manually requires navigating through multiple hidden system folders and stopping background services. This tool safely automates the entire process for you.

## Download
**[Download the latest release here!](https://github.com/l0wskilled/cs2-shader-cache-cleaner/releases/latest)** 
*(Just download the `CS2ShaderCleaner.exe` file)*

## How to use it
1. **Download** the `.exe` file from the link above.
2. **Right-click** the file and select **"Run as administrator"** *(This is required so the tool can safely manage services and restart your graphics driver)*.
3. The tool will automatically find your Steam folder.
4. Click **Start** and wait a few seconds.
5. The tool will automatically reopen Steam for you when it's done.

## What exactly does this do?
When you click start, the app will:
* Safely close Steam and CS2 if they are running.
* Clear out NVIDIA and Counter-Strike 2 cache files.
* Refresh your graphics driver configuration.
* Reopen Steam with normal user privileges.

No computer restart required.
